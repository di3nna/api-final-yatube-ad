# API для Yatube

## Описание
API для социальной сети Yatube. Позволяет создавать посты, комментарии, подписываться на авторов и просматривать контент. Аутентификация через JWT-токены.

## Установка
1. Клонируйте репозиторий.
2. Создайте виртуальное окружение и установите зависимости из `requirements.txt`.
3. Выполните миграции и запустите сервер.

## Примеры запросов
### Получение списка постов
GET /api/posts/

### Создание поста
POST /api/posts/
Authorization: Bearer <токен>
{"text": "Текст поста", "group": 1}
